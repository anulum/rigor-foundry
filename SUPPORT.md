# Support

Use the narrowest channel that matches the request:

- Usage and design questions: [GitHub Discussions](https://github.com/anulum/RIGOR-FOUNDRY/discussions)
- Reproducible defects: [GitHub Issues](https://github.com/anulum/RIGOR-FOUNDRY/issues)
- Security reports and private contact:
  [protoscience@anulum.li](mailto:protoscience@anulum.li)
- GitHub Security Advisories, when private vulnerability reporting is enabled:
  [private advisory form](https://github.com/anulum/RIGOR-FOUNDRY/security/advisories/new)

The project has no released support tier during migration. Do not send
credentials, proprietary source, private audit reports, or secret variable
values through issues or discussions.
